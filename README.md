# Table Booking App

> **Work in Progress** - This is a personal learning project and is actively being developed.

### Project Overview

This backend project is a REST API for a table booking application designed to help restaurants manage reservations. The application features CRUD operations and business logic for both staff and customer workflows.
Restaurant staff (admins) authenticate securely manage tables, table groups (e.g. "outside tables", "inside tables"), restaurant settings, opening hours, and bookings.
Customers can create bookings without authentication and receive an email containing a verification link to confirm their reservation.

This project was built with TypeScript, Express, SQLite, and Vitest for testing.
